**Improvements:**

* **Doc collection display names:** You can now give your doc collections custom display names to make them easier to identify at a glance. Set the display name when creating a doc collection, or edit it later in the settings modal. Without a display name, the doc collection shows the repository identifier (e.g., `acme/docs (github)`) as the default label.
