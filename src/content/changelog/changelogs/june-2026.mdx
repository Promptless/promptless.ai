**Improvements:**

* **Vale Prose Linting:** Promptless now lints prose with Vale when your docs repository includes a Vale configuration file (`.vale.ini` or `vale.ini`). Error-severity violations are fixed automatically before suggestions are created, and warning-level rules are evaluated against your established style. This catches style guide violations earlier in the review process. See [Vale Integration](/docs/configuring-promptless/doc-collections/vale-integration) for setup details.

**Bug Fixes:**

* **Suggestion Diff Editor Markdown Rendering:** Fixed bold formatting bleeding into subsequent lines when inline code appeared alongside bold text in the suggestion diff editor—markdown now renders correctly regardless of formatting combinations.
