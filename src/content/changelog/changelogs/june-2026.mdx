**Improvements:**

* **Persistent repository access button:** The **Manage repository access** button now appears at the bottom of the repository picker whenever you're selecting a docs repository—not just when no repositories are accessible. Click it to open GitHub's installation settings directly, where you can grant access to additional repositories or adjust which ones Promptless can reach.

**Bug Fixes:**

* **Suggestion Diff Editor Markdown Rendering:** Fixed bold formatting bleeding into subsequent lines when inline code appeared alongside bold text in the suggestion diff editor—markdown now renders correctly regardless of formatting combinations.
