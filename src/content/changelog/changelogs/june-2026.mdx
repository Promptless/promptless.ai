**Improvements:**

* **Self-explanatory Slack trigger links in PR descriptions:** Slack trigger links in docs PR descriptions now show clear labels that tell you where they lead—"Message in Slack channel #docs," "Private Slack DM with Jane Doe," or "Private Slack group message"—followed by a message snippet. For DMs and private group messages, the link is paired with a direct link to the suggestion's Triggers & Analysis tab so reviewers without Slack access can read the full message instead of hitting Slack's error page.

**Bug Fixes:**

* **Suggestion Diff Editor Markdown Rendering:** Fixed bold formatting bleeding into subsequent lines when inline code appeared alongside bold text in the suggestion diff editor—markdown now renders correctly regardless of formatting combinations.
