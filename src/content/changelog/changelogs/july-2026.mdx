**Bug Fixes:**

* **Slack link previews prompt a reconnect when a scope is missing:** Sharing a Promptless suggestion link in Slack renders a preview card, which needs the `links:write` permission. Workspaces connected before this permission was added couldn't render the card, and the link posted as plain text with no clear explanation. The Integrations page now flags an affected Slack connection as **Needs reconnect**, so you know to click **Reconnect** on the Slack card to grant the permission. Reconnecting keeps your existing connection, channel memberships, and configuration intact. See [Updating Slack Permissions](/docs/integrations/slack-integration#updating-slack-permissions).
