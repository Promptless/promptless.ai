Slite integration provides **read-only** access to your team's knowledge base for documentation automation. When Slite is configured as a context source, Promptless can search your Slite notes for relevant information when creating suggestions—but it never modifies your Slite content.

## How It Works as Context Source

When Slite is enabled as a context source:

- **Note Search**: Promptless searches your Slite workspace for relevant notes when creating documentation suggestions
- **Knowledge Context**: Internal documentation, product specs, and team knowledge from Slite inform new documentation
- **URL Resolution**: When triggers reference Slite note URLs, Promptless automatically fetches the linked note content

## Agent Capabilities

When Slite is connected, Promptless agents gain access to two specialized tools:

| Tool | Description |
|------|-------------|
| **Search Slite Notes** | Searches your Slite knowledge base by free-text query. Returns note IDs and titles for discovery. |
| **Get Slite Note** | Fetches a specific note by ID and returns its full markdown content, title, and URL. |

## Example

When a GitHub PR references a feature described in your Slite workspace, Promptless searches for related notes to understand additional product context. This ensures documentation includes accurate feature descriptions and terminology from your team's knowledge base.

## Configuration

Slite uses API key authentication rather than OAuth. To connect Slite:

1. In your Slite account, go to **Settings > API** and create a new API key.
2. We recommend creating the key from a dedicated service account so it can be revoked independently.
3. Go to the [Integrations page](https://app.gopromptless.ai/integrations) in your Promptless dashboard.
4. Click **Connect** on the Slite integration card.
5. Paste your API key in the modal and click Connect.

<Note>
The API key grants Promptless read access to all notes visible to the account that created the key. We recommend using a service account with appropriate access scope.
</Note>

## Data Privacy

Promptless queries Slite data in real-time and does not store any of your Slite content. All searches are performed on-demand when documentation updates are needed. Query results are cached briefly to improve performance during a single documentation session, then discarded.

Promptless only reads from Slite—it never creates, updates, or deletes notes.

## Disconnecting

To revoke Promptless's access to your Slite workspace:

1. Go to the [Integrations page](https://app.gopromptless.ai/integrations).
2. Click the Slite integration card.
3. Select **Disconnect**.

This removes the stored API key from Promptless. You should also revoke the API key in your Slite account settings to ensure no further access is possible.
